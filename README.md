# RestorationOps (Web App)

## 1) Run locally
1. Install Node.js (LTS)
2. Unzip this project
3. In a terminal inside the folder:
   - `npm install`
   - `npm run dev`
4. Open the URL shown (usually http://localhost:5173)

## 2) Deploy free on Vercel (fastest)
1. Create free accounts: GitHub + Vercel
2. Upload this folder to a new GitHub repo
3. In Vercel: New Project → Import Git Repository → Deploy
4. You’ll get a link like `https://your-app.vercel.app`

## 3) Install on iPhone
Open the link in Safari → Share → Add to Home Screen

## IMPORTANT
This scaffold includes UI components that match the imports used by your canvas code.
Replace `src/RestorationOpsMVP.tsx` with your full canvas file to get the full app UI.
